# Tarea2--Grupo5


¿Qué hace el programa?

Este programa tiene como objetivo calcular el Impuesto Sobre la Renta ISR que debe pagar una persona según su salario anual. El sistema aplica las tablas de tramos del ISR vigentes en República Dominicana para determinar el monto a pagar de forma automática.

¿Qué datos recibe de entrada?

El programa recibe como datos de entrada el salario mensual o anual de la persona. En caso de recibir el salario mensual, el sistema lo convierte a salario anual para poder aplicar los tramos correspondientes del ISR.

¿Qué datos procesa?

Una vez recibido el salario anual, el programa procesa la información comparándola con los rangos establecidos por la DGII. Calcula el excedente sobre el límite inferior del tramo, le aplica el porcentaje correspondiente y le suma la cuota fija de ese tramo para obtener el ISR anual. Luego divide entre 12 para sacar el ISR mensual.

¿Qué datos imprime?

El programa imprime en pantalla el salario anual, el salario mensual, el tramo de ISR que le corresponde, el monto de ISR anual y el monto de ISR mensual a retener.
